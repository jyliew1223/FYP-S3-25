# Screens

This folder contains screen components for your app.

## Structure:
- Each screen should be in its own folder
- Include screen component and related files

## Example:
```
screens/
├── Home/
│   ├── HomeScreen.js
│   ├── HomeScreen.styles.js
│   └── index.js
├── Profile/
│   ├── ProfileScreen.js
│   └── index.js
└── Auth/
    ├── LoginScreen.js
    ├── RegisterScreen.js
    └── index.js
```
