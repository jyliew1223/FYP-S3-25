# Images

This folder contains image assets for your app.

## Organization:
```
images/
├── icons/          # App icons, tab icons
├── logos/          # Brand logos
├── backgrounds/    # Background images
├── avatars/        # Default avatars
└── illustrations/  # Illustrations, graphics
```

## Best Practices:
- Use PNG for images with transparency
- Use JPEG for photos
- Use SVG for scalable graphics (with react-native-svg)
- Provide @2x and @3x versions for different screen densities
- Keep file sizes optimized

## Example naming:
```
icon-home.png
icon-home@2x.png
icon-home@3x.png
logo-primary.png
background-splash.jpg
```
