install gunicorn
    pip install gunicorn

generate requirements.txt
    pip freeze > requirements.txt

create a new file "Procfile" and paste:
    web: gunicorn GoClimb.wsgi --log-file -

install Django
    pip install django

Create new project
    django-admin startproject GoClimb

Create new app in GoClimb
    python manage.py startapp new app

    in GoClimb/settings.py add new app into Django
        INSTALLED_APPS = [
            "django.contrib.admin",
            "django.contrib.auth",
            "django.contrib.contenttypes",
            "django.contrib.sessions",
            "django.contrib.messages",
            "django.contrib.staticfiles",
            
            # Your App
            "new_app",
        ]


Django setup:
    In settings.py, set allowed hosts
    Replace your-render-service-url.onrender.com with the domain Render assigns your service.
        ALLOWED_HOSTS = ['your-render-service-url.onrender.com', 'localhost', '127.0.0.1']
       
    Set debug to False for production:
        DEBUG = False

    Ensure static files setup:
        STATIC_ROOT = BASE_DIR / 'staticfiles'

    and collect static files before deploy:
        python manage.py collectstatic
